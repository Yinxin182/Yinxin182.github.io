# Programmer Portfolio Website

A modern, responsive portfolio website for programmers/developers built with HTML, CSS, JavaScript, and Tailwind CSS.

## Features

- Responsive design for all devices
- Modern UI with animations and transitions
- Sections for About, Skills, Projects, Blog, and Contact
- Dark mode support
- Mobile-friendly navigation
- Smooth scrolling
- Form validation
